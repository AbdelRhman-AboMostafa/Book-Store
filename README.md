
# 📚 Book Store App

A simple and responsive Book Store web application built with Vue.js. It allows users to browse and explore books through a clean and modern interface.

---

## 🚀 Features

* Responsive design (mobile, tablet, desktop)
* Clean and modern UI
* Easy book browsing experience

---

## 🛠️ Tech Stack

* Vue.js
* JavaScript (ES6+)
* HTML5
* CSS3

---

## 📦 Installation

Clone the repository:

```bash
git clone https://github.com/your-username/book-store.git
```

Navigate to the project folder:

```bash
cd book-store
```

Install dependencies:

```bash
npm install
```

Run JSON file:

```bash
npx json-server db.json
```

Run the project locally:

```bash
npm run dev
```

---

## 🏗️ Build for Production

```bash
npm run build
```
